## Installation

<Tabs>
  <TabItem value="cli" label="CLI" default>
    ```bash npx nativeflare-ui add button ```
  </TabItem>
  <TabItem value="manual" label="Manual">
    ```jsx // button.js ```
  </TabItem>

## Usage

```jsx
import { Button } from '@/components/ui/button';
```

```jsx
<Button title="Button" variant="outline" />
```
